lis(1)
======

Name
----
lis - Daemon for controlling backlight brightness.


Synopsis
--------
'lis' [OPTION]...


Description
-----------
Auto dim/undim backlight when a user is idling/active.


Options
-------
*-c* <file>::
	path to config file. Default is '/etc/lis.conf'.

*-h, \--help*::
	display help and exit.
